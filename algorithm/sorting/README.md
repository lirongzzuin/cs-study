## 3. ВйћвЊю ВўѕВаю
### ­Ъћ╣ ВаЋваг ВЋїЖ│авдгВдў Жхгьўё
##### Java (ьђх ВаЋваг & в│ЉьЋЕ ВаЋваг)
```java
import java.util.Arrays;

public class SortingExample {
    public static void quickSort(int[] arr, int left, int right) {
        if (left < right) {
            int pivot = partition(arr, left, right);
            quickSort(arr, left, pivot - 1);
            quickSort(arr, pivot + 1, right);
        }
    }

    private static int partition(int[] arr, int left, int right) {
        int pivot = arr[right];
        int i = left - 1;
        for (int j = left; j < right; j++) {
            if (arr[j] <= pivot) {
                i++;
                swap(arr, i, j);
            }
        }
        swap(arr, i + 1, right);
        return i + 1;
    }

    private static void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }

    public static void main(String[] args) {
        int[] arr = {5, 3, 8, 4, 2};
        quickSort(arr, 0, arr.length - 1);
        System.out.println("ьђх ВаЋваг Ж▓░Ж│╝: " + Arrays.toString(arr));
    }
}
```

##### JavaScript (в▓ёвИћ ВаЋваг & ВѓйВъЁ ВаЋваг)
```javascript
function bubbleSort(arr) {
    let n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
    return arr;
}

let arr = [5, 3, 8, 4, 2];
console.log("в▓ёвИћ ВаЋваг Ж▓░Ж│╝:", bubbleSort(arr));
```

def merge_sort(arr):
    if len(arr) > 1:
        mid = len(arr) // 2
        left_half = arr[:mid]
        right_half = arr[mid:]
        merge_sort(left_half)
        merge_sort(right_half)
        
        i = j = k = 0
        while i < len(left_half) and j < len(right_half):
            if left_half[i] < right_half[j]:
                arr[k] = left_half[i]
                i += 1
            else:
                arr[k] = right_half[j]
                j += 1
            k += 1
        while i < len(left_half):
            arr[k] = left_half[i]
            i += 1
            k += 1
        while j < len(right_half):
            arr[k] = right_half[j]
            j += 1
            k += 1

arr = [5, 3, 8, 4, 2]
merge_sort(arr)
print("в│ЉьЋЕ ВаЋваг Ж▓░Ж│╝:", arr)

## 5. вЕ┤ВаЉ ВДѕвгИ ВўѕВІю в░Ј вІхв│ђ

### 1№ИЈРЃБ ВаЋваг ВЋїЖ│авдгВдўВЮё ВёаьЃЮьЋа вЋї Ж│авацьЋ┤ВЋ╝ ьЋа ВџћВєївіћ?

### 2№ИЈРЃБ ьђх ВаЋвагЖ│╝ в│ЉьЋЕ ВаЋвагВЮў В░еВЮ┤ВаљВЮђ?

### 3№ИЈРЃБ ВЋѕВаЋ ВаЋвагВЮ┤въђ вг┤ВЌЄВЮИЖ░ђ?
ВаЋваг ьЏёВЌљвЈё **Ж░ЎВЮђ Ж░њВЮ┤ ВъЁваЦвљю ВѕюВёювЦ╝ ВюаВДђьЋўвіћ ВаЋваг ВЋїЖ│авдгВдў**ВЮё ВЮўв»ИьЋювІц. Вўѕ: в│ЉьЋЕ ВаЋваг, ВѓйВъЁ ВаЋваг

### 4№ИЈРЃБ ьъЎ ВаЋвагВЮў ьі╣ВДЋВЮђ?
ьъЎ ВъљвБїЖхгВА░вЦ╝ ЖИ░в░ўВю╝вАю вЈЎВъЉьЋўвЕ░, **Вхювїђ ьъЎ вўљвіћ ВхюВєї ьъЎВЮё ВѓгВџЕьЋўВЌг ВаЋваг**ьЋювІц. ВхюВЋЁВЮў Ж▓йВџ░ВЌљвЈё O(n log n) Вё▒віЦВЮё ВюаВДђьЋювІц.

### 5№ИЈРЃБ O(n┬▓) ВаЋваг ВЋїЖ│авдгВдўВЮ┤ ВќИВаю ВюаВџЕьЋа Вѕў ВъѕвіћЖ░ђ?

### 6№ИЈРЃБ Ж│ёВѕў ВаЋваг(Counting Sort)ВЮў ВаюьЋюВаљВЮђ?
